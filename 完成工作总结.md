## 使用方法

1. **克隆代码并提交更改**
   ```bash
   # Linux/macOS
   chmod +x git_commit.sh
   ./git_commit.sh
   
   # Windows
   git_commit.bat
   ```

2. **在远程服务器上构建**
   ```bash
   git pull
   docker build -t mirrorbot-zh .
   docker run -d --name mirrorbot-zh -p 5244:5244 mirrorbot-zh
   ```

3. **查看 Alist 默认密码**
   ```bash
   docker logs mirrorbot-zh | grep -A 2 "初始管理员信息"
   ```

4. **访问 Alist**
   - 在浏览器中访问 `http://服务器IP:5244`
   - 使用用户名 `admin` 和获取到的密码登录
